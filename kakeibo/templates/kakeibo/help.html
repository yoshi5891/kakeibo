家計簿アプリの使い方

このアプリは、毎日の支出を簡単に記録し、月ごとの傾向を分かりやすく確認できるように作られています。

1. 支出を追加する

「支出を追加」ページから、以下の項目を入力して保存します。

  • 金額：半角数字で入力します。
  • 日付：自動で今日の日付が入ります。変更も可能です。
  • カテゴリ：最近使ったカテゴリが上に表示され、選びやすくなっています。
  • メモ：必要な場合のみ入力できます（折りたたみ式）。

2. ダッシュボードの見方

アプリのトップページでは、今月の支出状況をまとめて確認できます。

  • 今月の合計：今月の支出総額が表示されます。
  • 今日の支出：今日使った金額が表示されます。
  • 先月との比較：先月との差額が表示されます。
  • カテゴリ別円グラフ：どのカテゴリにどれだけ使ったかが一目で分かります。
  • 日別支出推移：1ヶ月の支出の流れを折れ線グラフで確認できます。
  • 前月 / 翌月ボタン：月を移動して過去のデータも確認できます。

3. 支出一覧

登録した支出を一覧で確認できます。

  • 編集：金額・カテゴリ・日付・メモを修正できます。
  • 削除：不要なデータを削除できます。
  • スマホ対応：横スクロールで全項目を確認できます。

4. 月別サマリー

カテゴリごとの支出を月単位で比較できます。

  • カテゴリ別の棒グラフ：どのカテゴリに多く使ったかが分かります。
  • 月ごとの推移：支出の変化を確認できます。

5. ログイン・ログアウト

ログインしている場合は、ナビバーに「こんにちは、〇〇さん」と表示されます。

ログアウトはメニュー内の「ログアウト」から行えます。

6. データのバックアップと復元

このアプリには、データを安全に保管するための バックアップ機能 と、過去の状態に戻すための 復元機能 が用意されています。

📦 バックアップの内容

バックアップを実行すると、次の内容が ZIP ファイルにまとめられ、GitHub の backups フォルダに保存されます。

  • 家計簿データ（data.json）
  • .env（存在する場合）
  • バックアップ作成日時（metadata.txt）
🔹 バックアップの作成方法（管理者のみ）
  • 画面下部の「バックアップ作成」ボタンを押す
  • または /backup_data にアクセスする
  • 数秒後、GitHub に ZIP が保存されます
🔹 データの復元方法（管理者のみ）

アプリには、万が一データが消えてしまったときに備えて、バックアップから元の状態に戻す機能があります。

  • 画面下部にある 「バックアップを復元」ボタン を押すと、保存されているバックアップの中から 一番新しいもの を自動で選びます。
  • アプリがバックアップの中身を読み取り、家計簿データを元の状態に戻します。
  • 復元が終わると、完了メッセージが画面に表示されます。

難しい操作は必要ありません。ボタンを押すだけで、アプリが自動で復元作業を行います。

⚠ 注意事項
  • バックアップの作成・復元は 管理者ユーザーのみ が行えます。
  • 復元を行うと、現在のデータはすべて上書きされます。必要なデータがある場合は、復元前にバックアップを作成してください。
  • バックアップ機能を使うためには、アプリに設定されている GitHub の接続情報 が正しく設定されている必要があります。
